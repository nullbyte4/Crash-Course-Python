## About the project

This repo tracks my hands-on work through **Crash Course on Python**, the first course of Google's *IT Automation with Python* Professional Certificate on Coursera — code exercises, small scripts, and a final project, written by me as I learn.

Conceptual notes, glossary, and open questions live separately in an Obsidian vault; this repo is strictly for code.

## Structure

No fixed layout yet. As exercises are added, they'll be organized per module (e.g. `03-loops/`), with this README and `CLAUDE.md` kept up to date to reflect it.

## How to run

Each exercise is a standalone script unless noted otherwise:

```bash
python path/to/exercise.py
```

Requires Python 3.x. No external dependencies expected for the early courses; this section will grow if that changes.
